Copyright, Creative Commons (CC) Licenses, Fair Use

Copyright protects original works of authorship, providing creators a standardized way to grant the public permission to use their creative work under copyright law.

Types of Creative Commons (CC) Licenses

CC BY: Attribution only.

CC BY-SA: Attribution, share-alike.

CC BY-ND: Attribution, no derivatives.

CC BY-NC: Attribution, non-commercial.

Fair Use Principles
  1. Purpose and character of use (e.g., commercial or educational).
  2. Nature of the copyrighted work.
  3. Amount and substantiality of the portion used.
  4. Effect on the market for the original work.

WEB SECURITY

Web Security

Common Cyber Attacks and Preventive Measures

  1. Phishing Attacks: Scrutinize the emails you receive, update passwords
  2. SQL Injection: Use an Intrusion detection system, Carry out a validation of the user-supplied data.
  3. Cross-Site Scripting (XSS): Mitigate XSS by sanitizing user inputs and using Content Security Policy (CSP) headers.
  4. DDoS Attacks: Implement rate limiting, use a Content Delivery Network (CDN), and have DDoS protection services in place.
  5. Malware Attack: Use antivirus software, Use firewalls, avoid clicking on suspicious links .
